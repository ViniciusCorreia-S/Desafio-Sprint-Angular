# DesafioAngular

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 19.2.7.

## Iniciar projeto

Como executar o projeto:

1- Abra um terminal na pasta do projeto Angular e digite:

npm start


2- Em outro terminal:

npm run api


3- Assim que o servidor estiver em execução, abra seu navegador e navegue até `http://localhost:4200/`. O aplicativo será recarregado automaticamente sempre que você modificar qualquer um dos arquivos de origem.
